# lbproxy
[![npm version](https://badge.fury.io/js/@copha%2Flbproxy.svg)](https://badge.fury.io/js/@copha%2Flbproxy)

A proxy server with load balance

## Install
```
npm i -g @copha/lbproxy 
# yarn global add @copha/lbproxy
```

## How to use
